# Roadmap — Aether

Lista de trabajo y mejoras; no forma parte de la descripción fija del proyecto.

## Corto plazo

- Vista previa 3D del personaje al personalizar (y equipar mallas según `CharacterDraft`).
- Pulir sliders y **CheckBox** en Ajustes (tema SAO coherente).
- Input de **mando** y navegación por el menú solo con teclado/gamepad.

## Medio plazo

- Tema UI más definido (fuentes, ilustración, SFX de foco).
- CI: export headless o comprobación de proyecto en push.
- Git LFS para assets pesados cuando el repo crezca.

## Largo plazo (visión)

- Distrito o región jugable única antes de escalar mundo.
- Sistemas de misión e inventario según el GDD.
